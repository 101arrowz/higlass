=======================
HiGlass Track Types
=======================

Bed-like
=================

.. image:: img/bedlike-track-thumb.png
    :align: right

The bedlike track is intended to display generic interval data. It is used to
render tracks with a `bedlike` datatype. This usually comes from the `beddb`
filetype.
